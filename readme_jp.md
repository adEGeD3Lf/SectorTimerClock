# SectorTimerClock 取扱説明書

**バージョン：1.22**

---

## 概要

SectorTimerClock は、デスクトップ上に常駐するアナログ時計アプリです。タイマーの残り時間を扇形（セクター）で視覚的に表示します。通常のタイマーのほか、複数のタイマーを連続して実行する「シーケンスタイマー」機能を備えています。

---

## 基本操作

### 移動

時計の文字盤を左クリックしたままドラッグすると、画面上の任意の位置に移動できます。

### タイマーの終了

右クリックメニューの「タイマー終了」を選択するか、タイマー終了後の点滅中に時計をクリックすると、タイマーを終了します。タイマー作動中の時計の文字盤をダブルクリックしてもタイマーを終了できます。

---

## 右クリックメニュー

時計を右クリックするとメニューが表示されます。

| 項目 | 説明 |
|------|------|
| タイマー終了 | 実行中のタイマーをリセットします |
| タイマー設定 | タイマー設定ダイアログを開きます |
| お気に入り | 登録済みのタイマー・シーケンスを即座に開始します |
| 日付の上下位置変更 | 日付表示の位置を上下に切り替えます |
| Language | 表示言語を日本語／English に切り替えます |
| 詳細設定 | 外観や動作の詳細設定ダイアログを開きます |
| 終了 | アプリを終了します |

---

## タイマー設定

右クリックメニューから「タイマー設定」を選択すると、以下のセクションを含むダイアログが開きます。

### 残り時間で設定

タイマーの残り時間を指定して開始します。

- 入力形式：`H:MM:SS`、`MM:SS`、または分数のみ（例：`1:30:00`、`45:00`、`10`）
- 全角数字・コロンにも対応しています
- **記憶**：入力値を履歴に保存します
- **削除**：選択中の履歴を削除します
- **お気に入りに追加**：現在の値をお気に入りに登録します
- **シーケンスに追加**：現在の値をシーケンスのステップに追加します
- **開始**：タイマーを開始します

### 時刻で設定

指定した時刻にタイマーが終了するよう設定します。

- 入力形式：`HH:MM:SS`（例：`15:00:00`）
- 指定時刻が現在時刻より前の場合は、翌日の同時刻として扱います
- ボタン類は「残り時間で設定」と同様です

### お気に入り管理

お気に入りに登録したタイマーを管理します。右クリックメニューから直接タイマーをスタートできます。

- **↑ 上へ／↓ 下へ**：選択中の項目を並び替えます
- **削除**：選択中の項目をお気に入りから削除します
- ドラッグ＆ドロップでも並び替えできます

### シーケンスタイマー

複数のタイマーを連続して実行する機能です。

#### 保存済みシーケンスの読み込み

1. 「保存済みシーケンス:」のドロップダウンから対象を選択します
2. **読み込み**ボタンを押すと、ステップ一覧に反映されます
3. **保存済み削除**ボタンで保存済みシーケンスを削除します（お気に入りにも登録されている場合は確認ダイアログが表示されます）

#### シーケンスの編集

- シーケンス名を入力します
- 「残り時間で設定」または「時刻で設定」の入力欄に値を入力後、**シーケンスに追加**ボタンでステップを追加します
- ステップ一覧で項目を選択し、以下のボタンで操作できます

| ボタン | 説明 |
|--------|------|
| ↑ 上へ | 選択中のステップを上に移動 |
| ↓ 下へ | 選択中のステップを下に移動 |
| １つのステップを削除 | 選択中のステップを削除 |
| ステップ一覧を削除 | すべてのステップをクリア |
| お気に入りに追加 | 保存済みシーケンスをお気に入りに登録 |

- ドラッグ＆ドロップでもステップを並び替えできます

#### 保存

- シーケンス名を入力し、**保存**ボタンを押すとシーケンスが保存されます
- 既存のシーケンス名と同じ名前の場合は**保存**ボタンがグレーアウトします（上書き不可）
- 保存済みのシーケンスと内容が完全に一致する場合のみ、**お気に入りに追加**ボタンがアクティブになります

#### シーケンスの実行

| ボタン | 説明 |
|--------|------|
| シーケンス開始 | ステップ一覧の先頭から実行します（ステップが空のときはグレーアウト） |
| ここからシーケンス開始 | 選択中のステップから実行します（2番目以降を選択したときにアクティブ） |

## シーケンスタイマーの動作

### ステップ間の移動

各ステップのタイマーが0になると、自動的に次のステップのタイマーが裏で開始します。同時にタイマー終了の点滅表示が行われます。

- 点滅中に時計をクリックすると点滅が止まり、次のタイマーの表示に切り替わります
- 点滅中も次のタイマーは動き続けています

### シーケンス終了

すべてのステップが終了すると、通常のタイマー終了と同じ点滅表示になります。時計をクリックするとリセットされます。

### 実行中の割り込み

シーケンス実行中に別のタイマーを開始しようとすると、「シーケンスタイマーが実行中です。中断しますか？」という確認ダイアログが表示されます。「中断する」を選ぶとシーケンスを終了して新しいタイマーを開始、「戻る」を選ぶとシーケンスをそのまま継続します。

### 時刻指定ステップの扱い

時刻指定のステップがシーケンスの途中にある場合、前のステップが終了した時点から「次の指定時刻に終了するタイマー」として起動します。指定時刻が既に過去の場合は翌日の同時刻として扱います。

---

## タイマーの表示

タイマーが動作中、残り時間に応じてモードが切り替わり、扇形の大きさと色で残り時間を表示します。

### モードの切り替え

| モード | 条件 | 表示の基準 |
|--------|------|------------|
| H モード | 残り1時間超 | 12時間の文字盤 |
| M モード | 残り1分〜60分 | 60分の文字盤 |
| S モード | 残り1秒〜60秒 | 60秒の文字盤 |

### 終了時の点滅

タイマーが0になると、設定した色で時計全体が点滅します。点滅の間隔は詳細設定で変更できます。

---

## 詳細設定

右クリックメニューから「詳細設定」を選択するとダイアログが開きます。

### 全体設定

| 項目 | 説明 |
|------|------|
| 常に最前面 | 他のウィンドウより手前に表示します |
| 秒針をスムーズにする | 秒針がなめらかに動きます（デフォルトは1秒刻み） |
| ウィンドウ位置を記憶する | 終了時のウィンドウ位置を次回起動時に復元します |
| 透明度 | 時計全体の透明度を設定します（10〜100%） |
| 直径 | 時計のサイズを設定します（50〜600px） |

### 文字盤の設定

- **数字**：色・フォント・文字盤からの距離を設定します
- **5分目盛り**：色・太さ・長さを設定します
- **1分目盛り**：色・太さ・長さを設定します
- **日付**：色・フォント・表示ON/OFF・透明度・上下位置・表示形式を設定します

### 針の設定

短針・長針・秒針それぞれについて、色・長さ・太さを設定します。

### タイマー表示

| 項目 | 説明 |
|------|------|
| H/M/S モードの色設定 | 残り時間に応じて扇形の色を変える段階設定を開きます |
| H/M/S モードの半径 | 扇形の大きさを設定します |
| リング | タイマー動作中に表示される外周リングの色と太さを設定します |
| 終了時の点滅 | 点滅の色と間隔を設定します |

#### 色段階設定

各モード（H/M/S）の「色設定」ボタンを押すと、残り時間に応じた色変化を設定するダイアログが開きます。

- 閾値（残り時間）を設定し、その時間以下になったときに適用する色を指定します
- **段階を追加**：新しい段階を追加します
- **段階を削除**：チェックボックスで選択した段階を削除します（最低1段階は必要）
- 閾値は自動的に降順で並び替えられます

---

## 設定ファイル

設定はアプリと同じフォルダの `settings.json` に自動保存されます。アプリを終了せずに設定変更した内容も即座にファイルへ書き込まれます。

### 起動時の注意

お気に入りに登録したシーケンスの元データ（保存済みシーケンス）が存在しない場合、起動時に自動でお気に入りから削除され、その旨が通知されます。

---

## ダイアログの位置

タイマー設定・詳細設定・色段階設定の各ダイアログは、時計と重ならない位置に自動的に表示されます。優先順位は「右 → 左 → 下 → 上」の順です。

---

## インストール

特別なインストール作業は不要です。ダウンロードしたZIPファイルを任意のフォルダに解凍するだけで使用できます。

## アンインストール

解凍したフォルダごと削除するだけです。レジストリや他の場所へのファイル書き込みは一切ありません。
