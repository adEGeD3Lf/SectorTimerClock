# SectorTimerClock V1.2

**SectorTimerClock** は、デスクトップに常駐するアナログ時計アプリです。タイマー機能を備えており、残り時間を扇形（セクター）でビジュアルに表示します。

## インストールと起動

1. ZIPファイルを任意のフォルダに解凍する
2. `SectorTimerClock.exe` をダブルクリックして起動

アンインストールはフォルダごと削除するだけです。レジストリは使用しません。

---

## 基本操作

| 操作 | 動作 |
|------|------|
| 左ドラッグ | 時計を画面上の任意の場所に移動 |
| 左ダブルクリック | タイマーをリセット |
| タイマー終了時に左クリック | 点滅を止めてリセット |
| 右クリック | メニューを開く |

ウィンドウ位置は自動で記憶され、次回起動時に同じ場所に表示されます（設定で無効化可能）。

## タイマー機能

タイマーは右クリックメニューの「タイマー設定」から起動します。

### 残り時間で設定

終了まで「何時間何分何秒」かかるかを指定する方式です。

入力例:
- `1:30:00` → 1時間30分後
- `45:00` → 45分後
- `10` → 10分後

### 時刻で設定

「何時何分に終わらせたい」という場合に使う方式です。指定した時刻がすでに過ぎていた場合は翌日の同時刻が設定されます。

入力例: `17:30:00` → 17時30分0秒まで

### タイマーの表示モード

残り時間の長さによって、扇形の表示スケールが自動で切り替わります。

| モード | 条件 | 扇形の1周 |
|--------|------|-----------|
| **Hモード** | 残り1時間超 | 12時間 |
| **Mモード** | 残り1〜60分 | 60分 |
| **Sモード** | 残り60秒以下 | 60秒 |

扇形は「現在位置から終了時刻まで」の残り時間を視覚的に表します。タイマー動作中は文字盤の外周にリングも表示されます。

### タイマー終了時

タイマーが終了すると、文字盤全体が指定した色で点滅します。点滅中に左クリックまたはダブルクリックするとリセットされます。

### 履歴とお気に入り

よく使う時間・時刻はコンボボックスに「記憶」ボタンで保存できます（最大20件、自動ソート）。さらに「お気に入りに追加」すると右クリックメニューから1クリックで即起動できます。お気に入りは順番の並び替えや削除が可能です。

**初期お気に入り（デフォルト）:**
- 残り5分
- 残り25分
- 時刻 12:00:00

## 右クリックメニュー

| 項目 | 説明 |
|------|------|
| タイマーリセット | 動作中のタイマーを即座にリセット |
| タイマー設定 | タイマー設定ダイアログを開く |
| （お気に入り一覧） | 登録済みのお気に入りを1クリックで起動 |
| 日付の上下位置変更 | 文字盤内の日付表示を上下に反転 |
| language | 言語を切り替える（日本語 / English） |
| 詳細設定 | 詳細設定ダイアログを開く |
| 終了 | アプリを終了 |

> **注意:** 言語を切り替えると設定が即座に保存されますが、UIへの反映は次回ダイアログを開いたときになります。

## 詳細設定

右クリック →「詳細設定」で開きます。設定ダイアログは時計ウィジェットと重ならない位置に自動で表示されます。

### 全体設定

| 項目 | 説明 |
|------|------|
| 常に最前面 | 他のウィンドウの上に常に表示する |
| 秒針をスムーズにする | 秒針を滑らかに動かす（デフォルトはカクカク） |
| ウィンドウ位置を記憶する | 終了時の位置を次回起動に引き継ぐ |
| 透明度 | 時計全体の不透明度（10〜100%） |
| 直径 | 時計の大きさ（50〜600px） |

### 文字盤の設定

**数字:** 色・フォント・文字盤からの距離を変更できます。

**目盛り:** 5分目盛りと1分目盛りそれぞれの色・太さ・長さを設定できます。

**日付表示:**

| 項目 | 説明 |
|------|------|
| 表示する | 日付表示のON/OFF |
| 透明度 | 日付テキストの透明度 |
| 距離（上下） | 中心からの上下オフセット（マイナスで上、プラスで下） |
| 形式 | 下記の4種類から選択 |

日付フォーマットの選択肢（日本語モード）:

| # | 表示例 |
|---|--------|
| 0 | 4/20 |
| 1 | 4/20(月) |
| 2 | 4月20日月曜日 |
| 3 | 2026/4/20 |

日付フォーマットの選択肢（英語モード）:

| # | 表示例 |
|---|--------|
| 0 | 4/20 |
| 1 | Mon 4/20 |
| 2 | Mon, Apr 20 |
| 3 | 4/20/2026 |

### 針の設定

短針・長針・秒針それぞれについて色・長さ・太さを設定できます。短針は長針より長くならないよう自動で制限されます。

### タイマー表示

**Hモード / Mモード / Sモード:** 各モードの扇形の色と半径（文字盤に対する割合）を設定します。

**リング:** タイマー動作中に外周に表示されるリングの色と太さを設定します。

**終了時の点滅:** 点滅色と点滅間隔（秒）を設定します。

---

## 設定ファイル

設定はすべて `settings.json` に自動保存されます。ファイルは `SectorTimerClock.exe` と同じフォルダに作られます。手動で編集することも可能ですが、通常はダイアログから操作するだけで問題ありません。

設定をリセットしたい場合は `settings.json` を削除して起動してください。
