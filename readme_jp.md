
# PomoSectorChrono

アナログ時計 × タイマーを融合した、デスクトップ常駐型のシンプルなタイマーアプリです。  
残り時間をアナログ時計の扇形（セクター）で直感的に可視化します。

---

## 特徴

- アナログ時計にタイマーの残り時間を**扇形（ウェッジ）**で重ねて表示
- タイマーは**残り時間指定**と**終了時刻指定**の2方式に対応
- 残り時間に応じて自動的に表示スケールが切り替わる（時間・分・秒モード）
- タイマー終了時は**点滅アニメーション**でお知らせ
- よく使うタイマー設定を**お気に入り**に登録してワンクリック起動
- 文字盤・針・色・サイズなど**外観を細かくカスタマイズ**可能
- ウィンドウ枠なし・常に最前面で**デスクトップに馴染む**デザイン

## 操作方法
いろいろ触ってみればだいたいわかると思いますが。
一応説明しておくと…

## 基本操作

| 操作 | 動作 |
|------|------|
| 左ドラッグ | 時計を移動 |
| 左ダブルクリック | タイマーリセット |
| タイマー終了後に左クリック | タイマーリセット |
| 右クリック | メニューを開く |

---

## 右クリックメニュー

| 項目 | 説明 |
|------|------|
| タイマーリセット | 実行中のタイマーを停止・リセット |
| タイマー設定 | タイマーの設定ダイアログを開く |
| 　残りXX:XX:XX / 時刻XX:XX:XX | お気に入り登録済みのタイマーをワンクリックで即スタート |
| 日付の上下反転 | 日付表示の位置を上下に切り替え |
| 詳細設定 | 外観などの詳細設定ダイアログを開く |
| 終了 | アプリを終了 |

お気に入りはタイマー設定の順番どおりに表示されます。登録がない場合は表示されません。

## タイマー設定

右クリック → **タイマー設定** から開きます。

### 残り時間で設定

カウントダウンする時間を指定します。入力形式は柔軟で、以下のいずれでも入力できます。

| 入力例 | 意味 |
|--------|------|
| `25` | 25分 |
| `45:00` | 45分 |
| `1:30:00` | 1時間30分 |

- **記憶** ボタン：入力値を履歴に保存します
- **削除** ボタン：選択中の履歴を削除します
- **お気に入りに追加** ボタン：現在の入力値をお気に入りに登録します（`残り01:30:00` のように保存）
- **開始** ボタン：タイマーをスタートします

### 時刻で設定

終了したい時刻を指定します（例：`17:30:00`）。  
指定時刻が現在時刻より前の場合は翌日の同時刻として扱われます。

- **お気に入りに追加** ボタン：`時刻17:30:00` のような名前でお気に入りに登録します

### お気に入り管理

タイマー設定ダイアログの下部に一覧が表示されます。

- **↑ 上へ / ↓ 下へ** ボタン：右クリックメニューでの表示順を変更できます
- **削除** ボタン：選択中のお気に入りを削除します

---

## タイマーの表示モード

残り時間によって扇形の表示スケールが自動で切り替わります。

| 残り時間 | モード | スケール |
|----------|--------|----------|
| 1時間超 | Ｈモード | 12時間スケール |
| 1〜60分 | Ｍモード | 60分スケール |
| 60秒以下 | Ｓモード | 60秒スケール |

タイマー終了後は時計全体が設定した色で点滅します。  
左クリックまたはダブルクリックでリセットできます。

## 詳細設定

右クリック → **詳細設定** から開きます。

### 全体設定

- **常に最前面**：他のウィンドウの前に常に表示します
- **秒針をスムーズにする**：秒針の動きをなめらかにします（CPU負荷がわずかに増加）
- **透明度**：時計全体の透明度を調整します
- **直径**：時計のサイズを変更します

### 文字盤の設定

- 数字の色・フォントを変更できます
- 数字の表示位置（半径）を調整できます
- 5分目盛り・1分目盛りそれぞれの色・太さ・長さを設定できます
- 日付表示の色・フォント・表示形式・透明度・上下位置を設定できます

### 針の設定

短針・長針・秒針それぞれの色・長さ・太さを変更できます。

### タイマー表示の設定

- H/M/Sモードそれぞれのウェッジ色と半径を変更できます
- リングの色と太さを変更できます
- タイマー終了時の点滅色と間隔を設定できます

---

## データの保存

設定・履歴・お気に入りはすべて `settings.json` に自動保存されます。  
ファイルはスクリプトと同じディレクトリに生成されます。
