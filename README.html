TIMER MODE

残り時間で自動切替。
3つのスケール。

タイマーをセットすると、残り時間に応じて扇形のスケールが自動で切り替わります。
長い時間でも短い時間でも、常に最適な見え方で。

H

Hモード

扇形1周＝12時間。長時間の作業計画に。

残り1時間超
M

Mモード

扇形1周＝60分。会議やポモドーロに。

残り1〜60分
S

Sモード

扇形1周＝60秒。ラストスパートを視覚化。

残り60秒以下
HOW TO SET

2通りのタイマー設定。

右クリック →「タイマー設定」から、残り時間か終了時刻を指定できます。

⏱ 残り時間で設定

1:30:00→1時間30分後
45:00→45分後
10→10分後

🕔 時刻で設定

17:30:00→17時30分まで
9:00→翌朝9時まで（過去なら）

よく使う設定はお気に入りに登録すると、右クリックから即スタートできます。

右クリックメニュー例： 残り00:05:00 残り00:25:00 時刻12:00:00 ← 1クリックで即スタート

SETTINGS

細かいところまで、
自分好みに。

サイズ・透明度直径50〜600px、透明度10〜100%でデスクトップに自然になじむ大きさに調整
針の色・太さ・長さ短針・長針・秒針それぞれ個別に設定。秒針をスムーズに動かすことも可
文字盤の数字・目盛りフォント・色・位置を変更。5分目盛りと1分目盛りを別々にカスタマイズ
日付表示4種類のフォーマット。上下位置・透明度も設定可。表示のON/OFFも可
タイマーの色と大きさH/M/Sモードそれぞれの扇形の色と半径を設定
終了時の点滅点滅の色と速さを設定
常に最前面他のウィンドウの上に常に表示するかどうかを切り替え
日本語 / English右クリックメニューの「language」から切り替え可能
INSTALL

インストールは解凍するだけ。

ZIPファイルをダウンロードする
任意のフォルダに解凍する
SectorTimerClock.exe をダブルクリックして起動

レジストリは使用しません。アンインストールはフォルダごと削除するだけです。
設定は settings.json に保存されます。リセットしたい場合はこのファイルを削除してください。
